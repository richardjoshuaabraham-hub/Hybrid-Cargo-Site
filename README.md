# BlueHarbor Hybrid Cargo - GitHub Pages Version

This version is ready for GitHub Pages.

## Files
- `index.html`
- `style.css`
- `script.js`

## Deploy with GitHub Pages
1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. Open the repository on GitHub.
4. Go to **Settings**.
5. Open **Pages**.
6. Under **Build and deployment**, select **Deploy from a branch**.
7. Choose the `main` branch and `/root`.
8. Click **Save**.
9. Wait a minute and GitHub will give you a public link.

Your site will be live at a `github.io` URL.
