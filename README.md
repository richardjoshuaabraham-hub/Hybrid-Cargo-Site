# BlueHarbor Hybrid Lines – GitHub Pages Version

## Upload to GitHub Pages

1. Create a GitHub repository.
2. Upload all files and folders from this project.
3. In GitHub, open **Settings > Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose **main** branch and **/root** folder.
6. Save.
7. Wait 1–2 minutes and open the published GitHub Pages link.

## Files to keep together
- index.html
- style.css
- script.js
- assets/logo-premium.svg

## Notes
- The site is interactive in the browser and works on GitHub Pages.
- Some photo panels use external image sources for realistic visuals. If a photo fails to load, the site shows a local fallback graphic instead.
